# SAL Task Management Application

## Project Description

SAL is a versatile task management application designed to help users efficiently manage their tasks and projects. With a user-friendly interface and powerful features, SAL makes it easy to stay organized and productive.

## Setup Instructions

1. **Clone the Repository**  
   Run the following command to clone the repository:
   ```bash
   git clone https://github.com/stevencito-jpg/SAL.git
   ```  

2. **Install Dependencies**  
   Navigate to the project directory and install the required dependencies:
   ```bash
   cd SAL
   npm install
   ```  

3. **Run the Application**  
   To start the application, use:
   ```bash
   npm start
   ```  

## Features

- **User Authentication:** Secure login and registration system.
- **Task Creation and Management:** Easily create, edit, and delete tasks.
- **Due Dates and Reminders:** Set deadlines for tasks and receive reminders.
- **Project Organization:** Group tasks into projects for better management.
- **Responsive Design:** Works seamlessly on both desktop and mobile devices.
